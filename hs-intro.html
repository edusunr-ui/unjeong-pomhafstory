외대HS&아르스썬 어학원에는 두 가지 형태의 클래스가 있습니다.
학생의 수준과 학습 성향에 따라 클래스를 분명하게 나누고, 각 반의 목표에 맞게 지도합니다.

Lecture Class

강의식 클래스

  • 레벨과 진도가 맞는 학생 중심 편성
  • 정규 커리큘럼 기반의 강의식 수업 운영
  • Daily Test와 T코칭으로 성취도 관리
Jump-Up Class

점프업 클래스

  • 소수 인원 중심의 밀도 높은 보정형 수업
  • 복습, 반복, 레벨 보정이 필요한 학생에게 적합
  • 숙제 검사, Daily Test, 리포트, 코칭까지 촘촘하게 운영

  3. 영작 말하기와 Writing

  5. Writing Test와 숙제 기록

Program Features 외대HS어학원 프로그램 특징

01

레벨별 맞춤 학습

레벨에 따른 학습자료를 활용한 본문과 Writing 학습을 기반으로 문장구조의 기초를 자연스럽게 습득하도록 지도합니다. 정확한 어순 배열 훈련으로 표현한 내용을 스스로 쓸 수 있게 만들고, 중등 내신과 서술형 대비의 초석을 다집니다.

02

E-Learning System

교재와 연계된 Voca, Listening, Sentence Building 온라인 학습으로 예습과 복습을 반복하며, 점진적인 난이도 상승을 통해 아이들이 배운 학습을 장기 기억으로 가져가도록 완전학습을 추구합니다.

03

평가 시스템

  • DT (Daily Test): 데일리 테스트
  • MMT (Mid-Month Test): 2주에 한 번, 학습 이해도를 확인하는 점검 테스트
  • MT (Monthly Test): 1달에 한 번, 학습 이해도를 확인하는 점검 테스트
04

T코칭

숙제 체크와 오답 점검을 바탕으로 1:1 코칭을 진행하며, 틀린 부분을 바로잡고 개별 피드백으로 수업 이해도를 다시 확인하는 시스템입니다.
