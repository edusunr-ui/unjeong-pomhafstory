English Program

영어 커리큘럼

외대HS어학원의 영어 커리큘럼은 초등부 파닉스/프라이머리 과정부터 중고등부 학기중 레벨별 클래스까지 단계적으로 설계됩니다. 강의식 클래스와 점프업 클래스를 학생 레벨과 학습 목적에 맞춰 운영합니다.
